SchedPro

SchedPro is a personalized class and activity scheduling system designed for Consolatrix College of Toledo City. It combines Google Classroom-style scheduling with Messenger-like group chats, helping students and teachers stay on track with their academic activities.

🚀 Features

📅 Class & Activity Scheduling – View and manage upcoming schedules.

💬 Subject-Based Group Chats – Communicate within subjects.

🔔 Real-Time Notifications – Stay updated on announcements and changes.

📝 Task & Event Reminders – Never miss an important activity.

🛠 Admin & User Roles – Teachers and students have different access levels.

🏗 Technologies Used

Frontend:

React.js (with Vite for fast development)

React Router (for navigation)

Tailwind CSS (for styling)

Backend:

Express.js (Node.js framework for APIs)

Postgresql (Database for schedules and chats)
