SchedPro 
Personalized Class and Activity Scheduling System for Students of Consolatrix College of Toledo City


     Overview
SchedPro is a web-based scheduling system that helps students and teachers organize their class schedules and activities efficiently. It integrates features similar to Google Classroom and Messenger, allowing seamless communication and scheduling in one platform.

     Features
 Class & Activity Scheduling – Easily plan and track schedules.
 Group Chat per Subject – Discuss assignments and announcements.
 Task Management – Mark assignments as Pending, In Progress, or Done.
 Notifications & Reminders – Stay updated with upcoming deadlines.
 Admin & Student Roles – Teachers manage schedules; students receive updates.

     Tech Stack
Frontend: React.js

Backend: Express.js + Postgresql

Authentication: JWT

Version Control: Git & GitHub
